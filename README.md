### 技术栈

- **Java**：JDK 21  
- **数据库**：MySQL 8.0  
- **服务器**：Tomcat 11.0.22  
- **前端**：HTML5 + CSS3 + JavaScript (JS) + JSP  
- **后端**：Java + Servlet + JavaBean + JDBC  
---

# 项目结构
Logistics-Management-System/  
├── database/schema.sql # 数据库脚本（含示例数据）  
├── pom.xml # Maven 依赖  
├── src/main/   
│  ├── java/com/logistics/  
│  │  ├── bean/ # JavaBean 实体  
│  │  ├── dao/ # JDBC 数据访问层  
│  │  ├── servlet/ # Servlet 控制器  
│  │  ├── filter/ # 编码/权限过滤器  
│  │  └── util/ # 工具类  
│  ├── resources/db.properties # 数据库配置  
│  └── webapp/ # JSP 页面 + 静态资源

## 前台用户端

| 模块       | 功能                                                                                     |
| ---------- | ---------------------------------------------------------------------------------------- |
| **快递查询** | 单号 + 公司查询、物流轨迹、复制/刷新、本地查询历史                                       |
| **用户系统** | 注册/登录、找回密码、个人中心（昵称/手机/密码）                                         |
| **我的快递** | 绑定单号、状态跟踪、删除、标记已签收                                                   |
| **快递公司** | 列表展示（名称/编码/电话/官网）                                                        |
| **公告/帮助** | 公告列表与详情、FAQ 帮助中心                                                         |

## 后台管理端

| 模块           | 功能                                                                               |
| -------------- | ---------------------------------------------------------------------------------- |
| **管理员登录**  | 权限控制，非管理员无法进入                                                        |
| **数据统计**    | 用户/快递/查询量、状态分布                                                        |
| **快递管理**    | 增删改查、按单号/状态筛选                                                        |
| **物流公司**    | 增删改、Logo 上传、启用/禁用                                                     |
| **用户管理**    | 查看用户、启用/禁用、查看绑定快递数                                               |
| **物流轨迹**    | 添加/删除轨迹，时间可自定义或自动生成                                             |
| **系统管理**    | 公告管理、修改密码、查询/登录日志                                                 |
